# parthivmenon.com

A curated portfolio of Parthiv Menon. Built with Next.js, Tailwind CSS, and an unhealthy amount of code revisions. It attempts to showcase the tinkerer in me.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Parthiv-M/portfolio.git
   cd portfolio
   ```

2. Install dependencies:

   ```bash
   npm ci
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Application routes and global styles.
- `src/components`: UI components organized by feature (landing, projects, work, and more).
- `src/data`: Contains `data.json` which drives the content of the portfolio.
- `src/hooks`: Custom React hooks.
- `src/types`: TypeScript interfaces and type definitions.
- `public`: Static assets including images and logos.
